# learning-network-automation
Documenting my journey learning CCNA networking + Python automation.

I'm a 3rd-year BSIT student documenting my journey toward becoming a Network Engineer, combining Cisco networking (CCNA) with Python automation. This repo is my public learning log — not polished, just progress.

Goal
Learn enough networking (CCNA) and Python to actually understand how my own network works — and get good enough at this to help others secure their networks too, not just my own.

Why I'm doing this?
What got me into this is finding it genuinely fun how networks form connections and enforce rules — and as a BSIT student, I wanted to apply what I'm learning in programming to that same space, not keep them separate.

Part of what makes this interesting to me is understanding how ad blocking and DNS filtering actually work — I've dealt with sketchy redirect popups (clicking a site and suddenly landing somewhere I never clicked) and want to understand and stop that at the network level, not just for myself, but to actually secure our home network for everyone using it.
